Lewis Joseph Feik Portfolio Website

Overview

This is a fast static portfolio built for internships. It is plain HTML, CSS, and JavaScript so it is easy to update in VS Code and easy to host on GitHub Pages.

What to edit first

• assets images avatar.png  Your headshot
• index.html  Main headline and intro
• resume.html  Program name, certifications, and work experience
• projects.html  Add new projects as you complete them

Local preview

Option 1 VS Code Live Server

• Install the Live Server extension
• Right click index.html and choose Open with Live Server

Option 2 Python simple server

• Open a terminal in the folder
• Run  python -m http.server 8000
• Visit  http://localhost:8000

GitHub Pages deploy

• Push the repo to GitHub
• In the repo settings, Pages, set Source to Deploy from a branch
• Select branch main and folder root
• Save

Notes about files

The PDFs and screenshots are stored in assets docs and assets images.
Keep file names stable once you share the link publicly so links do not break.
